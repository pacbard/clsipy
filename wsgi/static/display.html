The Common LaTeX Service Interface is a web service providing access to a LaTeX compiler.
This CLSI server runs on the OpenShift network as a free public service.

How to:

As img src use the url "http://domain.rhcloud.com/display?reqid=test_name&latex=Latex example:$\sum$".
The latex attribute contains latex code between \begin{body} and \end{body} .
For any question email me at alberto.anzellotti@gmail.com.
